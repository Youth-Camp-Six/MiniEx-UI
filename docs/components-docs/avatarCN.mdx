# 头像

## 类型
支持三种类型：图片、Icon 以及字符，其中 Icon 和字符型可以自定义图标颜色及背景色。
<ShowSouceCode
  element={
    <Flex x='start'>
      <Avatar>U</Avatar>
      <Avatar icon={<MiIcon.User/>}></Avatar>
      <Avatar src={img} round='round'></Avatar>
      <Avatar style={{ backgroundColor: '#fde3cf', color: '#f56a00' }}>U</Avatar>
      <Avatar style={{ backgroundColor: '#87d068' }} icon={<MiIcon.User style={{color:'yellow'}}/>} />
    </Flex>
  }
  code={`
    import { Avatar, MiIcon,Flex } from 'miniEx';
    import React from 'react';
    import img from '../assets/logo.png'

    const App: React.FC = () => (
      <Flex x='start'>
      <Avatar>U</Avatar>
      <Avatar icon={<MiIcon.User/>}></Avatar>
      <Avatar src={img} round='round'></Avatar>
      <Avatar style={{ backgroundColor: '#fde3cf', color: '#f56a00' }}>U</Avatar>
      <Avatar style={{ backgroundColor: '#87d068' }} icon={<MiIcon.User style={{color:'yellow'}}/>} />
    </Flex>
    );

    export default App;
    `
  }
  language="ts"
  />

## 尺寸
头像可自定义尺寸，两种形状可选。

<Avatar width="30px" height="30px" icon={<MiIcon.User></MiIcon.User>}></Avatar>
<Avatar width="40px" height="40px">U</Avatar>
<Avatar width="50px" height="50px" src={'https://gw.alipayobjects.com/zos/rmsportal/KDpgvguMpGfqaHPjicRK.svg'} />

## 展示代码

```js
import { Avatar, MiIcon } from 'miniEx';
import React from 'react';

// const url = 'https://gw.alipayobjects.com/zos/rmsportal/KDpgvguMpGfqaHPjicRK.svg';

const App: React.FC = () => (
  <Avatar width="30px" height="30px" icon={<MiIcon.User></MiIcon.User>}></Avatar>
    <Avatar width="40px" height="40px">U</Avatar>
    <Avatar width="50px" height="50px" src={'https://gw.alipayobjects.com/zos/rmsportal/KDpgvguMpGfqaHPjicRK.svg'} />
    <Avatar style={{ backgroundColor: '#fde3cf', color: '#f56a00' }}>U</Avatar>
);

export default App;
```

## 带徽标的头像
通常用于消息提示。

<Avatar status="busy" statusPosition="top-right">U</Avatar>
<Avatar status="online" statusPosition="bottom-right">U</Avatar>
<Badge value="15">
   <Avatar>U</Avatar>
</Badge>


## 展示代码

```js
import { Avatar, MiIcon } from 'miniEx';
import React from 'react';

const App: React.FC = () => (
  <Avatar status="busy" statusPosition="top-right">U</Avatar>
<Avatar status="online" statusPosition="bottom-right">U</Avatar>
<Badge value="15">
   <Avatar>U</Avatar>
</Badge>
);

export default App;
```
