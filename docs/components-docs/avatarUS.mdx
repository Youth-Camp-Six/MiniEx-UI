# Avatar

## Type
Image, Icon and letter are supported, and the latter two kinds of avatar can have custom colors and background colors.

<ShowSouceCode
  element={
    <Flex x='start'>
      <Avatar>U</Avatar>
      <Avatar icon={<MiIcon.User/>}></Avatar>
      <Avatar src={img} round='round'></Avatar>
      <Avatar style={{ backgroundColor: '#fde3cf', color: '#f56a00' }}>U</Avatar>
      <Avatar style={{ backgroundColor: '#87d068' }} icon={<MiIcon.User style={{color:'yellow'}}/>} />
    </Flex>
  }
  code={`
    <Flex x='start'>
      <Avatar>U</Avatar>
      <Avatar icon={<MiIcon.User/>}></Avatar>
      <Avatar src={img} round='round'></Avatar>

      <Avatar style={{ backgroundColor: '#87d068' }} icon={<MiIcon.User style={{color:'yellow'}}/>} />
    </Flex>
    `
  }
  />

## Avatar.Group
Avatar group display.

  element={AvatarGroupCN()}

      const options = [
        {
